Swallow Project : Using xgdb

Useful gdb commands

Inspect instrs pointed to in lr
x/4i $lr-4

set breakpoint: "break <label>"
0x000108bc

Show ten lines around pc
x/20i $pc-20

Print pc in hex
p/x $pc
